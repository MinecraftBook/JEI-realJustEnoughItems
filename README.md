# JEI-realJustEnoughItems

> 现实版 Just Enough Items —— 地球Online 的配方查询系统

## 这是什么？
Minecraft 有 [JEI (Just Enough Items)](https://github.com/mezz/JustEnoughItems) 帮你查合成配方，现实世界却没有。  
本项目想改变这一点。

## 愿景
- 🔍 输入任意物品，查看它的"制作方法"和"能做什么"
- 📜 数据来源：《天工开物》《本草纲目》《齐民要术》等古籍（公有领域）
- 🌍 覆盖生活各领域：工艺、医药、农业、烹饪...
- 🌐 多语言支持：中文起步，社区翻译扩展
- 💻 双版本：网页即开即用 + 本地离线部署

## 灵感来源
本项目灵感来源于 Minecraft 模组 **JEI (Just Enough Items)** by mezz  
原仓库：https://github.com/mezz/JustEnoughItems
